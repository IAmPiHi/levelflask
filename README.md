# 🚀 LevelUp - 自我驅動紀錄 WebApp

這是一款專為自我成長設計的工具，旨在透過「遊戲化」的方式，將日常任務轉化為成長動力。你可以透過紀錄待辦事項獲取「經驗值」，讓進步變得肉眼可見！

[**🌐 立即線上試用**](https://levelsystem.onrender.com)

---

## 🌟 核心特色

- **經驗值系統 (XP)**：完成任務後給予經驗值回饋，提供精神上的正向激勵。
- **成長里程碑**：詳細記錄學習或訓練的歷程，打造屬於自己的成長日記。
- **輕量化設計**：基於 Flask 後端與原生前端技術，反應迅速且介面直覺。

## 🚀 本地開發環境設置

若要在本地端執行此專案，請按照以下步驟操作：

### 1. 複製專案
```
git clone https://github.com/IAmPiHi/levelflask.git
cd 檔案位置
```

### 2. 安裝必要套件
請確保你已安裝 Python，接著安裝 Flask：
pip install flask

### 3. 啟動伺服器
python app.py

啟動後，請在瀏覽器造訪：http://127.0.0.1:5000
